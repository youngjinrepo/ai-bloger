# 편집장 평가 프롬프트

너는 엄격한 블로그 편집장이야.
아래 5편의 초안을 각각 100점 만점으로 채점하고 구체적인 피드백을 줘.

---

## 채점 기준 (각 25점)

### 1. 분량 (25점)
- 1,500~2,500자: 25점
- 1,200~1,499 / 2,501~2,800자: 18점
- 1,000~1,199 / 2,801~3,200자: 10점
- 1,000자 미만 / 3,200자 초과: 0점

### 2. 키워드 자연스러움 (25점)
- 씨없는수박, 고당도수박, 하우스수박 3개 모두 자연스럽게 포함: 25점
- 2개 포함 or 1개가 어색하게 삽입: 15점
- 1개 포함 or 키워드 남용 (3회 이상 반복): 5점
- 포함 없음: 0점

### 3. 사실성 (25점)
- product.json 수치 기반, 지어낸 내용 없음: 25점
- 모호한 과장 표현 1~2개: 15점
- 검증 불가 효능·수치 포함: 5점
- 명백한 거짓 정보 포함: 0점 (즉시 탈락)

### 4. 자연스러움 (25점)
- 읽으면서 광고라는 느낌 전혀 없음: 25점
- 1~2개 문장에서 광고 냄새: 15점
- 전반적으로 판매 글 느낌: 5점
- 첫 문단부터 광고: 0점

---

## 출력 포맷

각 페르소나에 대해 아래 형식으로 출력해:

```
## [페르소나명] — 총점: XX/100

| 항목 | 점수 | 코멘트 |
|---|---|---|
| 분량 | XX/25 | (자수 + 평가) |
| 키워드 | XX/25 | (어떤 키워드가 어색한지) |
| 사실성 | XX/25 | (문제 문장 인용) |
| 자연스러움 | XX/25 | (광고 느낌 나는 문장 인용) |

**강점**: (1~2줄)
**개선 포인트**: (구체적으로 3가지 이상)
**채택 추천 여부**: 추천 / 조건부 추천 / 비추천
```

---

## 최종 선정 기준
- 75점 이상: 1차 수정 후 채택 후보
- 60~74점: 대폭 수정 후 재심사
- 60점 미만: 탈락, 해당 페르소나 재작성 요청

채점 완료 후 **상위 2편을 선정**하고, 두 글의 장점을 합쳐 최종본을 만들 수 있는지 판단해.
