# 최종 출력 프롬프트

편집장이 최종 선정한 글을 기반으로 아래 포맷으로 완성본을 출력해.

---

## 출력 포맷

### 1. 제목 후보 (3~5개)

조건:
- 메인 키워드 1개 이상 포함
- 25자 이내 (모바일 검색 결과에서 잘림 방지)
- 광고성 문구 금지 ("강추", "역대급" 등)
- 독자의 페인포인트 또는 궁금증을 건드리는 형식

```
제목 1:
제목 2:
제목 3:
제목 4: (선택)
제목 5: (선택)
```

---

### 2. 본문

완성된 본문 전체 출력.
`(총 XXX자)` 표기.

---

### 3. 사진 삽입 가이드

| 위치 | 사진 파일 | 설명 (블로그 업로드 시 alt/캡션) |
|---|---|---|
| 도입부 상단 | screenshot_1.png | 수토마켓 스윗수박 — 11브릭스 이상 고당도 씨없는 수박 |
| (해당 문단 번호) | screenshot_3.png | 비파괴 당도측정기 실측 수치 11.8 Brix |
| (해당 문단 번호) | screenshot_4.png | 씨앗 선발부터 바이오포장까지 10단계 수박 프로세스 |
| 마무리 직전 | screenshot_6.png | 수박 최적 보관 온도 10°C 안내 |

※ 사진이 불필요한 위치는 제외. 최소 2장, 최대 4장.

---

### 4. 태그 (10개 이상)

조건:
- 검색량 있는 키워드 우선
- 브랜드명 포함
- 시즌 태그 포함 (여름, 제철 등)
- 네이버 블로그 기준 #태그 형식

```
#씨없는수박 #고당도수박 #하우스수박 #수박택배 #제철과일
#산지직송수박 #수토마켓 #스윗수박 #여름과일 #당도보장수박
#수박추천 #봄수박 #수박11브릭스 #과일배달 #여름간식
```

(위는 예시. 글 내용에 맞게 조정할 것)
