# 수정 요청 프롬프트

편집장의 피드백을 받았어. 아래 피드백을 반영해서 글을 수정해.

---

## 수정 규칙

1. **피드백에 명시된 문장만 수정해** — 나머지는 건드리지 마
2. **키워드를 추가할 때는 문장을 자연스럽게 재구성해** — 키워드를 그냥 끼워 넣지 말 것
3. **분량 조정이 필요하면** — 정보 가치 낮은 문장 삭제 (감성 과잉 문장, 중복 내용 우선 삭제)
4. **사실성 문제가 있으면** — 해당 문장을 product.json 기반으로 교체하거나 삭제

---

## 수정본 제출 포맷

```
## 수정 전 (원문 발췌)
"..."

## 수정 후
"..."

## 수정 이유
(편집장 피드백 중 어느 항목을 반영했는지)
```

수정 완료 후 전체 글 다시 출력해.
총 자수도 함께 표기: `(총 XXX자)`
